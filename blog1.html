Getting started with Web Development

How the Website works?

Before starting your journey in becoming a Web Developer ,you must first know how the website works.

When you type a web address into your browser,the browser goes to the DNS server,and finds real address of server that website lives on.

The browser then sends an HTTP request to the server, the server if approves the request ,it sends resposnse in form of copy of the website to the client, this response is in form of HTML, CSS and Javascript and our browser interprets and displays it for us.

So it's necessary to first learn basics of website building i.e. HTML, CSS and Javascript .

What is HTML?

HTML is the basic necessity a developer should know before building a website. Once you learn HTML,you can make simple websites easily.

HTML stands for HyperText Markup Language. It is used to design web pages using a markup language. Hypertext defines the link between the web pages. A markup language is used to define the text document within tag which defines the structure of web pages.

HTML is a markup language that is used by the browser to manipulate text, images, and other content to display in the required format.



Below is example of simple HTML code:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
</body>
</html>

What is CSS?

CSS stands for Cascading Style Sheet.
Where HTML is what defines the structure and content of a web page, a Cascading Style Sheet is a web document that allows you to change the appearance of the HTML.

CSS is a stylesheet language used to design the webpage to make it attractive. The reason of using CSS is to simplify the process of making web pages presentable.

CSS plays like a magic. See with same HTML format, we can create different webpages and make all of them unique just by adding different styling. CSS allows us to create beautiful website for attracting audience.



Below is example of CSS styling:

body {
     background-color: blue;
}
h1 {
     color: red;
     padding: 60px;
}

What is Javascript?

JavaScript is the world most popular programming language.
It is also known as JavaScript for web pages.

JavaScript is well-known for the development of web pages. JavaScript can be used for Client-side developments as well as Server-side developments.

JavaScript helps us to add functionality our website. It helps users to interact with the website. For eg. clicking on buttons, working on user's input to give desired output , etc.



Example:

<script>

console.log("WELCOME");

</script>
